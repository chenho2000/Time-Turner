World time map

World time map (DOM manipulations) show current times for all time zone. The utility allowed user to enter date and time if they want (Note here it will user's current timezone), or it will shows the current time for each time zone. If they clicked on each individual time zone, it will shows the time at that time zone in the bottom area. Users can choose the format of showing times. They need to reclick on that map in order to see the changed format. User can click on the Digit Time/ Clock button to choose if they want to see the digit number or a clock. Also they need to click on the map again to see the clock. If users choose to entered the date by their own. This option will not be avalibale. The clock option will only avaliable for the live time. They can click "reset" button to reset to current time. It's a useful tool if the web developer want to shows all of the time more clearly at the same time. For example: a flight app may need this utility to shows the time in different time zone country of a flight's take off or landing time.
