Time Object Methods Examples

Constructor
(time1 will be used for most of the method examples below [which is your current time])
(all other Time object used below are created from here)

  • current time with current timezone:
    const time1 = new Time();
  • Setting time with a dateString with current timezone:
     const time2 = new Time("July 7, 2020 00:00:00");
  • Setting time with a date String with given TimeZone (last number):
     const time3 = new Time("July 7, 2020 00:00:00", -4);
  • Setting time with given exact information with current time zone:
    (year, MONTH INDEX (0 for Jan, 1 for Feb ....), day, hours, minutes, seconds, milliseconds)
     const time4 = new Time(2020, 6, 7, 0, 0, 0, 0);
  • Setting time with given exact information with given time zone (last number):
    (year, MONTH INDEX (0 for Jan, 1 for Feb ....), day, hours, minutes, seconds, milliseconds, timezone)
     const time5 = new Time(2020, 6, 7, 0, 0, 0, 0, -4);

setTime() method:

  • set year:


    time1.setTime("y", 2019);

  • set month:


    time1.setTime("M", 7);

  • set day:


    time1.setTime("d", 7);

  • set hour:


    time1.setTime("h", 0);

  • set minutes:


    time1.setTime("m", 0);

  • set seconds:


    time1.setTime("s", 0);

  • set timezone:


    time1.setTime("t", 8);

  • After setting it will become:

    (time1.format("yyyy-MM-dd hh:mm:ss");)

convertTime() method:

  • Here we change the current time in your timezone to (GMT +8) , After Converting it will return a Time object with converted time
    const Converted = time1.convertTime(8);

    The converted time will be: (Converted.format("yyyy-MM-dd hh:mm:ss");)

getUTCtime() method:

  • Here we change the current time in your timezone to UTC time , After Converting it will return a object with converted time
    const Converted = time1.getUTCtime();

    The converted time will be: (Converted.format("yyyy-MM-dd hh:mm:ss");)

DOM manipulations:

Constructor (Read the instructions in the examples)

  • World Time Map::
    const map = new WmapGenerator();

    Add the world map to the DOM:
    map.addAll();
  • Simple Timezone converter:
     const converter = new converterGenerator();

    Add the Timezone converter to the DOM:
     conveter.addAll();
  • Major city time visual representation:
     const cityTime = new CTGenerator();

    Add the Major city time representation to the DOM:
     cityTime.addAll();
  • Major country time visual representation:
     const countryTime = new CapGenerator();

    Add the Major country time visual representation to the DOM:
     countryTime.addAll();
  • StopWatch:
     const stopwatch = new StopWatchGenerator();

    Add the StopWatch to the DOM:
     stopwatch.addAll();
