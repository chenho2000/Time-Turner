Getting Started

First: Download the library js file

  • You can click "Download" on the navbar to download "Time-Turner.zip" file

    Decompress the file you will get a folder named "Time-Turner" which has src folder and the Time-Turner.js (which is the library js file) in it. You can choose to move the whole Time-Turner folder or the src folder and the js file to your workplace. But please notice the src folder and the js file must stay together (in the same path). (Strongly recommend to move the folder since you may already have a src folder for your project in your workplace) No external library is required to install for Time-Turner.

Second: Setup the library to HTML

  • After finishing the step above. You can load the js library to your HTML file(s). What you need to do is put this line of code to your HTML file in the header. (Path may be different)
  • <script> defer type="text/javascript" src="./Time-Turner.js" </script>
  • Notice here the src path is where you put your Time-Turner.js.

Start using the library

  • You have finished all the steps to set up the Time-Turner library. In order to use the library correctly please read the instruction below and look at the example page.

Time Object

Constructor

  • Time Object can take up to 8 arguments and there are different ways to create the Time Object. All of the way to create the Object is list below. Please read the following carefully.
  • current time with current time zone:
    const time1 = new Time();
  • Setting time with a date String with current time zone:
     const time2 = new Time("July 7, 2020 00:00:00");
  • Setting time with a date String with given time zone (last number):
     const time3 = new Time("July 7, 2020 00:00:00", -4);
  • Setting time with given exact information with current time zone:
    (year, MONTH INDEX (0 for Jan, 1 for Feb ....), day, hours, minutes, seconds, milliseconds)
     const time4 = new Time(2020, 6, 7, 0, 0, 0, 0);
  • Setting time with given exact information with given time zone (last number):
    (year, MONTH INDEX (0 for Jan, 1 for Feb ....), day, hours, minutes, seconds, milliseconds, time zone)
     const time5 = new Time(2020, 6, 7, 0, 0, 0, 0, -4);

Property

  • date : A Date() Object
    Type: Date()
  • timezone: the time zone number
    Type: Number

Methods

  • format(a) method:
    Argument:
    (String) format method take 1 argument which can be any combinations with any symbol between them of ("y","M","d","h","m","s","t") which represent (year, Month, day, hour, minute, second, time zone) For example : yyyy-MM-dd hh:mm:ss
    Return value:
    (String) format method will return a string represent the time with the format that user chose.
     format Time in different ways (yyyy represent years, MM represent month, ss represent seconds the symbol or combination between them are up to users)
  • setTime(a, b) method:
    Argument:
    (String, Number) setTime(a,b) method take 2 arguments, the first one represents the (y, M ,d ,h, m ,s ,t) users want to change. The second one represents which number user want it to be. (Notice here it is not allowed to change the time zone less than -11 or greater than 12)
     Return value:
    Nothing. The Time object's date/time zone will changed
     Set Date information (Year (y), Month(M), Date(d), Hours(h), Minutes(m), Seconds(s), Time zone(t))
  • getTime(a) method:
    Argument:
    (String) getTime(a) method take 1 arguments, represent the (y, M ,d ,h, m ,s ,t) users want to get.
     Return value:
    (Number/String) It will return Number except for time zone, for time zone it will return "GMT" + the time zone number
     Get Time information (Year (y), Month(M), Date(d), Hours(h), Minutes(m), Seconds(s), Time zone(t))
  • convertTime(a) method:
    Argument:
    (Number) convertTime(a) method take 1 arguments, represent the time zone number user want to change.
    Return value:
    (Time()) It will return a new Time() object with converted time and given time zone. (Note here it will return null if user enters invalid time zone number. (greater than 12 or less than -11)
     Convert Date and time between different Time zone return a new Time() Object
  • getUTCtime() method:
    Argument:
    No argument needed for this method
    Return value:
    (Time()) It will return a new Time() object with UTC-0 Time with UTC-0 timezone
     Get the UTC time of the given Time Object return a new Time() Object
  • getWeekDay() method:
    Argument:
    No argument needed.
    Return value:
    (String) It will return a string represent the current weekday of the given Time Object
     Get weekdays information (Monday- Friday)
  • add(a, b) method:
    Argument:
    (Number, String) add(a,b) method take 2 arguments, The first one represent which number user want it to be. (Notice here it's not allowed to have the time zone less than -11 or greater than 12) it will log(not available) to the console. the second one represent the (M ,d ,h, m ,s ,t) users want to add.
    Return value:
    No return value. It will add to the time of the given Time
     Calculation (add days/hours/minutes/seconds/time zone)
  • subtract(a, b) method:
    Argument:
    (Number, String) subtract(a,b) method take 2 arguments, The first one represent which number user want it to be. (Notice here it's not allowed to have the time zone less than -11 or greater than 12) it will log(not available) to the console. the second one represent the (M ,d ,h, m ,s ,t) users want to subtract.
    Return value:
    No return value. It will subtract the time of the given Time
     Calculation (subtract days/hours/minutes/seconds/time zone)
  • nextWeekday(a) method:
    Argument:
    (String) nextWeekday(a) take one argument, which is the string of the weekday user want to get. (notice here the first letter must be upper case and the rest must be lower case ex: Monday)
    Return value:
    (Time()) A new Time Object which is the next weekday user want to get
  • timeBetween(a, b) method:
    Argument:
    (Time(), Srting) timeBetween(a,b) take two argument. The first one is the Time object user want to compared. If it's not the same time zone as user's it will convert to the same first. The second argument is the unit user want to see. It can be in (days, hours, minutes) (d ,h, m)
    Return value:
    (Number) The number of (d ,h ,m) between a and b
    Time between current time object and another time object , compared in (days, hours, minutes)
  • beginningOf(a) / endOf(a) method:
    Argument:
    (String) beginngOf/endof(a) take one argument. It represent the end of a. a can be d for day, h for hour.
    Return value:
    (Number) It will return a number represent how many minutes between now and the beginngOf/endof user's choice. (h/d). It will also log a message to the console as well
     Time between now(the given Time() object) and the start/end of day/hour (output are in minutes)
